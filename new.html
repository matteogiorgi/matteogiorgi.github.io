Hello World!

I'm Matthew, a computational tinkerer with a strong foundation in mathematics, computer science, and finance. Holding a BSc in Computer Science from the University of Pisa, I'm currently enrolled at the Department of Mathematics of the University of Padua, pursuing an MSc in Computational Finance, as an aspiring quant.

My academic interests lie at the intersection of numerical methods and mathematical programming, with a particular focus on stochastic optimization and portfolio management. Additionally, I maintain a keen interest in cryptanalysis and, since the beginning of my studies, I have been passionate about programming languages and compiler construction.

Experiences

  • IMC-CHALLENGE First Italian team and ranked 108th worldwide in the 2025 edition of IMC Prosperity Challenge, a 15-day international algorithmic trading competition with more than 12,000 participants. Teaming up with three other University of Padua graduates in Computational Finance and Statistics, we implemented robust Python algorithms, tackled probability brainteasers, and delivered a top national performance, culminating in a +155K profit during the final round.
  • TOODY Designed and developed a Python web application from scratch to analyze software requirements documents and detect variability indicators. Built as part of my BSc thesis, the project covers both backend and frontend, using Flask for the web interface and spaCy for natural language processing.
  • LAB-ASSISTANT Worked, in support of Professor Pelagatti, as lab assistant for the C laboratory exam at the BSc in Physics at the University of Pisa. Helped students during exercise sessions and graded assignments.
  • PODEEN Public repository containing configuration files and installation scripts for a complete and efficient minimal UNIX work environment based on Debian or derivatives.
  • ASTEROIDS JavaScript implementation of the classic game Asteroids using p5.js. Useful to understand the basics of event-driven programming.
  • WIENER-REPORT Independent studies, under the supervision of Professor Romani (University of Pisa), on attacks exploiting vulnerabilities of the RSA cryptosystem, with a focus on the Wiener Attack and its use of continued fractions for factoring the RSA modulus.
  • FUNINT Designed and implemented an OCaml interpreter for a toy language with static scoping and dynamic type checking that handles tuples of expressions and can invoke tuples of functions for composition.
  • WORDLE Developed a Java implementation of the New York Times Wordle. Focused on efficient string comparison for word matching in a classic client–server structure.
  • GRAPH Implemented a Java undirected graph library for homogeneous generic objects. Strengthened object‑oriented design and modularity, emphasizing code reuse and separation of concerns.
  • MEMBOX Developed a concurrent C server for a virtual repository system, capable of handling non‑null byte sequences with synchronization primitives to ensure safe access in multi‑threaded environments. Focus on robust memory management and concurrent data structures (POSIX threads).
  • SPARSE Implemented a simple C library to handle sparse matrices efficiently. Valuable practice in pointer management, improving memory efficiency and low‑level programming skills.
  • COMPUTABILITY Draft notes and examples on computability theory (Turing machines, recursive functions, undecidability). Though incomplete, it provided exposure to theoretical CS and formal languages.
  • CNR-PROJECT Worked with a colleague as a Python developer at the CNR offices in Pisa on a scale model for blind people, supervised by Dr. Furfari (CNR) and Prof. Pelagatti (University of Pisa).
  • HACKATHON First place at ContHackt 2021 (University of Pisa & Contamination Lab). Our team later joined the 2021/22 EUAcceL program and won the final stage with a blockchain prototype for food supply‑chain tracking.
