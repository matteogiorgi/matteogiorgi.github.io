Major projects

Repositories

  • pde-base: base configuration for a personal Debian development environment.
  • pde-plug: Vim and VSCode plug-ins installers for your Debian development environment.
  • pde-tile: tile manager for your Debian development environment using Qtile.
  • toody: variability detector web application written for my BSc thesis.
  • wordle: Java implementation of the popular game Wordle.
  • asteroids: Javascript implementation of the popular game Asteroids.
  • computability: failed attempt to write comprehensive notes on the theory of computability.
  • wiener: an article on the RSA cryptosystem and the attack M.J. Wiener proposed in 1990.
  • funint: simple interpreter written in Ocaml for a toy language with static scoping and dynamic type checking.
  • graph: an implementation of undirected graph of generic homogeneous objects with no loops nore multiple edges.
  • membox: concurrent C server implementing a virtual repository hosting non null sequences of bytes.
  • sparse: C library that allows to handle sparce matrices.
